Compilation and Installation

This package includes the source files of a stand alone library tgraph and the editor pigale.
The editor is based on the Qt toolkit and uses the OpenGL libraries for the 3-dimensional drawings.
g++ and Mesa3D are availaible under the GNU license.
Qt is free under Unix with a GPL license, and also under Windows with a non-commercial license.

  • g++ compiler: g++-2.96 or g++-3.xx (GCC home page)
  • Qtlibrary: 3.0 (at least 2.3.1), with openGL support (Qt home page)
  • OpenGL : Mesa3D-3.4. (Mesa 3D home page)
  • glut libraries (OpenGl)
  • Doxygen (Documentation tool). (Doxygen home page)

Linux

The Mesa 3D libraries are included in XFree86 version 4, but sometimes the glut library is not.
The QT libraries are included in all Linux distributions providing access to KDE.
Doxygen is only needed if you want to create the online documentation.

The files pigale/GNUMakefile,pigale/tgraph/GNUMakefile , pigale/ClientServer/GNUMakefile and pigale/qt/GNUmakefile might need to be edited for particular goals (level of optimization, location of files, etc.).

The command make (GNU make) creates the libraries in pigale/lib and the executable programs in pigale/qt and pigale/ClientServer.

If you don't have root permission, you can create static versions of Pigale by typing: make static , after the libraries has been created (name is then spigale_xxx instead of pigale_xxx).

  • If the variable OPT is set to dbg (debugging mode) the resulting programs is named pigale_dbg.
  • If the variable OPT is set to opt (optimize mode) the resulting programs is named pigale.

The client/server programs are created with Client and Server names.

If you only need the tgraph library, you must first copy dummy.cpp into tgraph, then, in the repertory tgraph, type make.

This program has also been compiled under different UNIX systems, using the gcc compilers.

Windows

We used the following:

  • Microsoft compiler,
  • The non-commercial Qt toolkit,
  • The OpenGL libraries included in Windows,
  • The glut libraries.

To run the program, you need to install glut32.dll and qt-mt230nc.dll (or the qt dll corresponding to your license).
